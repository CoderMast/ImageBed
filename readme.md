## 项目介绍

本项目是基于 SpringBoot3 + Vue3 开发的一款前后端分离的图片管理系统，又称为图床系统，可以对图片进行管理。

项目主要使用到的技术有：Java + SpringBoot + Vue + MySQL + Nginx + Maven

项目主要分为：系统设置、用户管理、图片管理、存储库管理这四大主要模块。
